# Proyecto WAF Testing - Redes de Computadores

## Descripción General

Sistema integral de análisis y pruebas de seguridad para Web Application Firewall (WAF). Este proyecto implementa una aplicación web con capacidades de monitoreo de seguridad, simulación de ataques cibernéticos y análisis detallado de logs del WAF de Cloudflare.

---

## Características Principales

### Análisis Avanzado de Seguridad
- Detección y documentación de ataques SQL Injection (SQLi)
- Identificación de vulnerabilidades Cross-Site Scripting (XSS)
- Monitoreo en tiempo real de intentos de acceso no autorizados
- Registro detallado de eventos de seguridad con timestamps precisos

### Monitoreo de WAF
- Integración con Cloudflare WAF para protección de aplicaciones
- Análisis de headers de respuesta HTTP de seguridad
- Seguimiento de IDs de bloqueo y rayos CF
- Estadísticas en tiempo real de ataques detectados y bloqueados
